# The modified version of Chatty Cathy by Dev Dungeon

## Directions
1. Make a `.env` file
2. Copy paste this into the `.env file`
```
TOKEN=
CHANNEL=
DATABASE=db.sqlite3
```
3. "TOKEN" put your bot token
4. For "CHANNEL" put your __channel id__
